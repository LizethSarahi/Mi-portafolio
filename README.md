# Portafolio Adaptable (Responsive) con Bootstrap 5

Este proyecto fue creado para el bootcamp Tecnolochicas PRO, es una página web adaptable a dispositivos de distintos tamaños (este tipo de sitio web se conoce en inglés como "responsive"). 

El propósito de esta página web es mostrar mi portafolio de proyectos como desarrolladora y mi experiencia. Incluye una descripción breve de mi motivación, proyectos y formas de contacto. 


Esta página web fue creada con:

* HTML
* CSS
* JavaScript 
* Bootstrap 5

Además, se incluyeron **Google Fonts** para personalizar la fuente y **Bootstrap icons** para incorporar íconos como flechas y logos de redes sociales populares.
